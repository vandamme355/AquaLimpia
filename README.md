# Proyecto AquaLimpia S.A.

## Objetivo
Analizar el desempeño de las plantas de tratamiento para detectar incumplimientos en parámetros de calidad del efluente.

## Datos
Dataset con variables operacionales como caudal, DBO de entrada y salida, consumo energético y lodos generados.

## Proceso
- Exploración de datos
- Limpieza de valores faltantes
- Cálculo de eficiencia del tratamiento
- Análisis por planta
- Visualización mediante gráficos

## Resultados
Se identificaron diferencias en el desempeño entre plantas, evidenciando posibles desviaciones en la eficiencia del tratamiento.

## Conclusión
El análisis permite apoyar la toma de decisiones mediante la identificación de patrones operacionales y riesgos de incumplimiento.
